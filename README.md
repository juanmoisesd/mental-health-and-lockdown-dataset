## Description

Dataset on mental health indicators during COVID-19 lockdown. Compiles data on the psychological impact of lockdown measures, including anxiety, depression, and psychological well-being indicators across different populations during the pandemic.

## Overview
This repository contains data and resources related to **mental health and lockdown dataset**. It is part of an open science initiative to share research findings and datasets with the global scientific community.

## Research Context
The project addresses key questions in the field of neuroscience and social sciences, focusing on providing accessible data for further analysis and validation.

## Repository Structure
- `data/`: Contains the datasets used in this research.
- `src/`: Source code for data processing and analysis.
- `results/`: Output files, figures, and metrics.
